Wie wird der Signal-Score berechnet?

Der Signal-Score misst, ob relevante Entwickler ein Repo gestarrt haben — nicht nur wie viele.

FollowerGewichtBedeutung
500–9991 PunktAktiver Entwickler
1.000–4.9992 PunkteBekannter Entwickler
5.000–9.9994 PunkteEinflussreicher Entwickler
10.000+8 PunkteTop-Developer / Thought Leader

Notable Density = Score ÷ (Stars / 1.000) — normalisiert fuer Repo-Groesse.
Hoher Wert = proportional viele relevante Stargazer. Niedriger Wert bei vielen Stars = moegliches Red Flag.

Stichprobe: je 100 frueheste + 100 neueste Stargazers pro Repo (max. 200 von allen).
